<h1 align="center"> Projeto Spring :leaves: </h1>

#### Tecnologias usadas:

    - Java 8
    - Maven 3.6
    - Spring Framework
    - h2 Database
    - Swagger2 (API docs)
    - ModelMapper

##### Como usar:
    $ git clone
    $ mvn install
    $ mvn spring-boot:run
